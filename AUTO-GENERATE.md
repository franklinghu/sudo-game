# ✨ 自动生成练习题功能说明

**功能**：学生完成答题后，可以点击按钮自动生成明天的练习题

---

## 🎯 功能特点

### 1️⃣ 一键生成
- 答题完成后点击「✨ 生成明日练习」按钮
- 选择需要生成的科目
- 系统自动生成新的答题页面

### 2️⃣ 支持科目
- ⚡ 物理
- ⚖️ 道法
- 📖 语文
- 🔢 数学
- 📗 英语
- 📜 历史
- 🧪 化学

### 3️⃣ 自动生成
- 自动下载新的 HTML 文件
- 文件名格式：`2026-03-20-physics.html`
- 包含 5 道新题目和答案

---

## 📱 使用步骤

### 步骤 1：完成今日答题
1. 打开今日的答题页面
2. 完成所有题目
3. 查看成绩和解析

### 步骤 2：生成明日练习
1. 点击「✨ 生成明日练习」按钮
2. 在弹窗中选择科目（如：物理）
3. 等待生成（约 1 秒）

### 步骤 3：下载文件
- 文件自动下载到本地
- 文件名：`2026-03-20-physics.html`

### 步骤 4：上传到 GitHub
1. 访问：https://github.com/Franklinghu/zhongkao-practice
2. 点击 **Add file** → **Upload files**
3. 上传下载的 HTML 文件
4. 点击 **Commit changes**

### 步骤 5：完成！
- GitHub Pages 会自动更新（1-2 分钟）
- 新的答题页面即可访问

## 📁 文件命名规范

生成的文件遵循以下命名规则：

```

示例：
2026-03-19-tuesday.html    (周二综合)
2026-03-20-physics.html    (物理)
2026-03-20-ethics.html     (道法)
2026-03-20-chinese.html    (语文)
```

## 🔧 技术说明

### 题目生成
- 内置题库（每科 5+ 道题）
- 随机抽取（可扩展）
- 包含答案和解析

### 文件生成
- 纯前端生成（无需服务器）
- 使用 Blob 和 URL.createObjectURL
- 自动触发下载

### 数据存储
- 答题记录保存在 LocalStorage
- 错题本自动更新
- 关闭浏览器不丢失

---

## ⚠️ 注意事项

### 1. 题目重复
- 当前版本题目固定
- 多次生成同一科目会得到相同题目
- 建议：每科每天生成一次

### 2. 文件管理
- 下载的文件保存在浏览器默认下载目录
- 建议：创建文件夹分类保存
- 上传到 GitHub 后可删除本地文件

### 3. GitHub 更新
- 每次上传后等待 1-2 分钟
- GitHub Pages 会自动部署
- 清除浏览器缓存查看最新版本

## 📞 需要帮助？

遇到问题时：

1. **文件无法下载**：检查浏览器是否阻止了下载
2. **题目不显示**：清除浏览器缓存后重试
3. **上传失败**：检查 GitHub 网络连接
4. **页面不更新**：等待 2 分钟或强制刷新（Ctrl+F5）
