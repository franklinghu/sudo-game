🚀 GitHub 部署助手

中考答题系统 - 3 分钟部署到网络

⚠️ 部署说明

由于需要 GitHub 账号认证，请按以下步骤手动操作。

预计用时：3 分钟

目标仓库：https://github.com/Franklinghu/zhongkao-practice

1 创建 GitHub 仓库

1. 访问 GitHub 创建页面：

🔗 打开 GitHub

2. 填写以下信息：

  • Repository name: zhongkao-practice
  • Description: 中考 100 天冲刺 - 每日答题系统
  • 选择 Public（公开）
  • ✅ 勾选 Add a README file

3. 点击 Create repository

2 上传答题页面

1. 在仓库页面点击 Add file → Upload files

2. 打开文件管理器，进入：

/home/admin/.openclaw/workspace/daily-practice/

3. 将文件拖拽到上传区域：

2026-03-18-monday.html

4. 在下方输入提交信息：Add Monday practice

5. 点击 Commit changes

3 启用 GitHub Pages

1. 点击仓库的 Settings 标签

2. 左侧菜单点击 Pages

3. 在 Build and deployment 下设置：

  • Source: Deploy from a branch
  • Branch: 选择 main
  • Folder: 选择 / (root)

4. 点击 Save

4 等待部署

等待 1-2 分钟，页面刷新后会显示：

Your site is live at: https://franklinghu.github.io/zhongkao-practice/

✅ 答题链接：

https://franklinghu.github.io/zhongkao-practice/daily-practice/2026-03-18-monday.html
✅ 部署完成检查清单

部署完成后，在浏览器中测试以下功能：

  • 页面能正常打开
  • 可以切换四科标签页（语文/历史/数学/英语）
  • 可以点击选项
  • 显示正确答案和解析
  • 成绩统计正常
  • 错题本功能可用
