## 📖 使用说明

### 如何答题
1. 在下方「每日练习」列表中点击日期链接
2. 在浏览器中打开 HTML 答题页面
3. 在规定时间内完成所有题目
4. 系统自动判分并显示答案和解析
5. 错题自动加入错题本

### 功能特点
✅ **在线答题** - 直接在浏览器中答题  
✅ **自动判分** - 选择答案后立即显示对错  
✅ **题目解析** - 每题都有详细解析  
✅ **错题本** - 自动记录错题，支持查看解析  
✅ **成绩统计** - 显示总分、正确率、用时  
✅ **进度追踪** - 实时显示答题进度
