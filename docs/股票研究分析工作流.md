已安装的股票研究分析技能，支持 A 股/港股/美股的全流程分析。

## 🔄 标准研究流程

### 1️⃣ 初步筛选
```
确定研究范围 → 行业/板块筛选 → 基本面初筛
```
**指令示例：**
- "筛选半导体行业的股票"
- "找出市盈率低于 20 的科技股"
- "分析新能源板块的资金流向"

### 2️⃣ 深度分析
```
实时行情 → 技术分析 → 财务分析 → 资金流向
```
**指令示例：**
- "分析贵州茅台 (600519.SH) 的股价走势"
- "查看腾讯控股 (0700.HK) 的技术面"
- "分析宁德时代的财务报表"

### 3️⃣ 对比评估
```
同行业对比 → 估值对比 → 成长性对比
```
**指令示例：**
- "对比比亚迪和特斯拉的估值"
- "比较招商银行和平安银行的财务指标"
- "分析白酒板块的龙头股"

### 4️⃣ 投资决策
```
风险评估 → 买卖建议 → 仓位配置
```
**指令示例：**
- "给出 AAPL 的投资建议"
- "评估这只股票的风险等级"
- "帮我配置一个科技股投资组合"

### 5️⃣ 持续跟踪
```
价格预警 → 财报提醒 → 新闻监控
```
**指令示例：**
- "设置 AAPL 的价格预警"
- "监控特斯拉的财报日期"
- "追踪半导体行业的热点新闻"

## 📊 核心分析维度

### stock-analysis（8 维度分析）

| 维度 | 权重 | 说明 |
|------|------|------|
| 盈利惊喜 | 30% | EPS 超预期程度 |
| 基本面 | 20% | P/E、利润率、成长性 |
| 分析师情绪 | 20% | 评级、目标价 |
| 历史表现 | 10% | 过往财报反应 |
| 市场环境 | 10% | VIX、大盘趋势 |
| 板块表现 | 15% | 相对强度 |
| 动能指标 | 15% | RSI、52 周区间 |
| 市场情绪 | 10% | 恐惧贪婪、做空比例 |

### china-stock-analysis（A 股/港股）

- 实时行情（收盘价、涨跌幅、成交量）
- 技术面分析（支撑/压力位、均线）
- 资金流向（大单净流入、龙虎榜）
- 投资建议（买入/持有/卖出）

### akshare-stock（A 股量化）

- 实时行情查询
- 历史 K 线数据（日/周/月）
- 财务数据（报表、指标）
- 板块/行业分析
- 资金流向
- 龙虎榜数据
- 新股/IPO 信息
- 融资融券

## 🛠️ 常用指令

### 美股分析
```bash
# 基础分析
uv run scripts/analyze_stock.py AAPL

# 快速模式（跳过内部交易和新闻）
uv run scripts/analyze_stock.py AAPL --fast

# 对比多只股票
uv run scripts/analyze_stock.py AAPL MSFT GOOGL

# 加密货币
uv run scripts/analyze_stock.py BTC-USD ETH-USD
```

### A 股分析
```python
# 实时行情
import akshare as ak
stock_zh_a_spot_em()

# 历史 K 线
stock_zh_a_hist(symbol="000001", period="daily", 
                start_date="20240101", end_date="20241231", adjust="qfq")

# 财务数据
stock_financial_abstract_ths(symbol="000001", indicator="按报告期")

# 资金流向
stock_individual_fund_flow(stock="000001", market="sh")
```

### 投资组合管理
```bash
# 创建投资组合
uv run scripts/portfolio.py create "科技组合"

# 添加资产
uv run scripts/portfolio.py add AAPL --quantity 100 --cost 150

# 查看组合
uv run scripts/portfolio.py show
```

### 预警设置
```bash
# 添加股票到观察列表（带目标价）
uv run scripts/watchlist.py add AAPL --target 200

# 设置止损预警
uv run scripts/watchlist.py add AAPL --stop 150

# 检查触发的预警
uv run scripts/watchlist.py check
```

### 热点追踪
```bash
# 热门股票扫描
python3 scripts/hot_scanner.py

# 传闻扫描（早期信号）
python3 scripts/rumor_scanner.py
```

## ⚠️ 注意事项

1. **数据延迟**：Yahoo Finance 可能延迟 15-20 分钟
2. **内部交易数据**：SEC  filing 延迟 2-3 天
3. **做空比例**：FINRA 数据延迟约 2 周
4. **A 股数据**：AkShare 接口可能因目标网站变动而失效
5. **免责声明**：所有分析仅供参考，不构成投资建议
