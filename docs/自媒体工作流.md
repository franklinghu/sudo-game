## 🔄 标准工作流

### 1️⃣ 选题调研
```
搜索最近热点 → 分析竞品内容 → 确定选题方向
```
**指令示例：**
- "搜索最近 AI 工具的热点话题"
- "分析竞品在小红书上的爆款内容"

### 2️⃣ 内容策划
```
确定目标平台 → 选择内容形式 → 制定大纲
```
**指令示例：**
- "帮我策划一个小红书系列选题，主题是 AI 效率工具"
- "生成一个公众号文章大纲，讲自媒体运营技巧"

### 3️⃣ 内容创作
```
生成标题 → 撰写正文 → 优化 SEO → 添加 CTA
```
**指令示例：**
- "写一篇小红书笔记，主题是'3 个 AI 工具让效率翻倍'"
- "写一个抖音口播脚本，介绍 OpenClaw 技能"
- "优化这个标题，让它更有点击率：XXX"

### 4️⃣ 视觉制作
```
生成封面图 → 制作信息图 → 添加二维码
```
**指令示例：**
- "生成一个小红书风格的封面图，主题是 AI 工具"
- "做一个数据可视化的信息图"

### 5️⃣ 发布与优化
```
多平台发布 → 数据监控 → 复盘优化
```
**指令示例：**
- "分析这篇内容的数据表现"
- "生成周报，总结本周内容运营情况"

## 🚀 快速开始

### 新手模式（单篇内容）
```
1. "帮我写一篇小红书笔记，主题是 [你的主题]"
2. "生成一个配套的封面图"
3. "优化标题和标签"
```

### 进阶模式（系列内容）
```
1. "搜索 [领域] 的热点话题"
2. "策划一个 5 篇的系列选题"
3. "写第一篇，风格要 [专业/轻松/种草]"
4. "生成配套的图片"
5. "写一个发布计划"
```

### 团队模式（全流程）
```
1. "分析竞品账号的内容策略"
2. "生成月度内容日历"
3. "批量创作 10 篇内容"
4. "安排发布计划"
5. "每日数据监控和周报"
```

## 📝 内容模板

### 小红书笔记模板
```
{emoji} {数字 + 关键词 + 痛点} 标题

{开头钩子：制造好奇/共鸣}

{正文：短段落 + emoji + 换行}

{总结/行动号召}

{#标签 5-10 个}
```

### 公众号文章模板
```
{吸引人的标题}

{开头：痛点/故事引入}

## 小标题 1
{内容}

## 小标题 2
{内容}

## 小标题 3
{内容}

{结尾：升华 + CTA}
```

### 抖音脚本模板
```
【时长】XX 秒
【开头 3 秒】{留住观众的钩子}
【正文】{核心内容，口语化}
【结尾】{引导点赞评论}
【文案】{发布时的描述}
【标签】{#hashtags}
```

---

## ⚠️ 注意事项

1. **内容真实性**：所有数据引用必须可验证，不生成虚假评价
2. **避免绝对词**：不用"最好"、"第一"、"唯一"等违规词
3. **SEO 检查**：发布前用 seo 技能检查关键词布局
4. **平台规范**：遵守各平台内容规范，避免违规
